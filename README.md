## 模型总结 | Model Summary

- ✅ **线性回归** - 最基础的模型，用于回归任务。
- ✅ **卷积神经网络 (CNN)** - 图像识别、分类的主流模型。
- ✅ **长短期记忆网络 (LSTM)** - 处理时间序列数据的经典模型。
